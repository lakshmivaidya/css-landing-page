# HTML/CSS Landing Page

# Project Overview
This project is a responsive landing page created using HTML and CSS, based on the provided Figma design. The goal of this task is to accurately convert the given design into a functional and responsive web page that works seamlessly across all devices.

# Task Description
- Create a single landing page using only HTML and CSS
- Follow the design provided in the Figma file
- Ensure the page is fully responsive for desktop, tablet, and mobile devices

# Figma Design Link
https://www.figma.com/community/file/1405477150071965596  


# Tech Stack Used
- HTML5
- CSS3


# Features
- Pixel-accurate layout based on Figma design  
- Responsive design for all screen sizes  
- Clean and semantic HTML structure  
- Organized and reusable CSS
